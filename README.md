# LegalAI

LegalAI is an AI-powered legal drafting and research platform for Indian Advocates.

## Goals

- Legal Notice Generator
- Plaint Generator
- Written Statement Generator
- Agreement Generator
- Affidavit Generator
- Case Law Research
- Bare Acts Search
- AI Legal Assistant

## Technologies

- CSS
- JavaScript
- Python
- FastAPI
- SQLite
