# Legal Drafting System

A modern Legal Drafting and Judicial Assistant Tool designed for Advocates, Law Students, and Legal Professionals.

## Features (Planned)

- Legal Templates
- Petition Drafts
- Vakalatnama
- PR Bond
- Bail Applications
- Legal Notices
- Citation Library
- Supreme Court Judgments
- Bombay High Court Judgments
- IPC to BNS Converter
- Acts & Sections
- Powerful Search
- PDF Export
- Offline Support

## Tech Stack

- HTML
- CSS
- JavaScript
- Supabase
- GitHub
